To check the PCB board design, 
use the Sparkfun.dru file with the DRC tool.

Then use the cmthunes-gerber-excellon.cam file
 with the cam tool to generate gerber files. 
 (Use process job).


Files to put in zip file for Gerber file:
 .gbl
 .gbo
 .gbs
 .gtl
 .gto
 .gts
 .drd
 
 
 Use this website to view the Gerber Files after zipping:
 http://www.gerber-viewer.com/
